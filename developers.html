Changes

This plugin is a modification of the original Ushahidi Heatmap plugin by dkiboa. It can be found here

We made the following modifications to this plugin:

    views/heatmap/sidebar.php
  • Embedded Heatmap into mainpage as a seperate map
  • Brought the heatmap button to the top of the sidebar
  • Added toggle function that allows users to switch between main map and heatmap
  • Refactor sidebar.php to seperate map handling js into its own file (heatmap.js)
    • views/heatmap.js
  • Changed the draw behaviour of heatmap
  • Added code to handle map updating
    • openlayers_heatmap.js
  • Added blue overlay to map
  • Brought the heatmap button to the top of the sidebar
    • controllers/heatmap.js
  • Load retrived data into a json file
